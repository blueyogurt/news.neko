ペットロス症候群とは

ペットロス症候群は、文字通り「ペットを失った悲しみ」によって心身的に起こる症状のことです。

■ペットロス症候群の症状

出典 : http://karadanote.jp/12218

ペットロス症候群は、ペットを失ってしまった悲しみが受け入れられないために起こるストレスで引き起こされるものです。
症状としては、以下のようなものが挙げられています。

・深い落ち込み
・何にも関心・興味が持てなくなる
・何も楽しめない、喜べない
・常にだるい
・不眠
・食欲不振
・集中できない
・やる気がでない
・自責感、罪責感

これらの症状は多少なりともペットを失った方に起こるものですが、この症状のせいで日常生活が困難になるほどであればそれは「ペットロス症候群」だと思って間違いないでしょう。

ペットロス症候群を克服するために、まずは自分の内面を知ろう

ペットロス症候群はいわゆる「気持ち」の部分の整理がつかないために起こる病気です。
自分が今どんな気持ちなのか、なぜペットロス症候群になってしまったのかを考えることは克服への大切な一歩となります。

■人が感じる悲しみのプロセスを知る

出典 : http://inulog.net/owakare/155.html

ペットを失ってしまったときに感じる悲しみはとても深いものです。
だからこそ、その悲しみがどのようなものなのか理解しておくことが大切です。

出典 : http://nekogazou.com/pet20141125/

人の悲しみは「拒否・怒り・交渉・抑うつ・受容」の5段階で回復するという説があります。

ペットを失ったと信じられない気持ち(拒否)
なぜ助けられなかったのかと悔やむ気持ち(怒り)
もしペットが戻ってくるなら何でもするという気持ち(交渉)
ペットを失って何をする気力もない状態(抑うつ)

この4つを経たあと、段々と気持ちが楽になってくる(受容)と言われています。

出典 : http://www.osohshiki.jp

ペットがいなくなってしまったことを受け入れられるまでの期間は人それぞれです。
長い期間悲しい気持ちから抜け出せないとき、人はペットロス症候群になってしまいます。

ペットロス症候群になる原因を考える

なぜペットロス症候群になってしまったのか、自分の心に問いかけてみましょう。

■今までペットに頼りすぎていませんでしたか？

出典 : http://bashoh.com

ペットは家族の一員。
時には誰よりも心強い存在となってくれることもあったでしょう。
しかし、過度の依存はペットロス症候群に陥りやすくなってしまいます。

■ペットに対して罪の意識がありませんか？

出典 : http://ja.wikipedia.org

「あのときこうしてあげていれば…」「もっと○○してあげればよかった」など、自分を責めていませんか？
大きすぎる責任感は、ペットロス症候群を引き起こしやすくなります。

■周りの人に悲しみを理解してもらえますか？

出典 : http://www.gatag.net

たかがペットじゃない」と言われたり、周りに悲しみを打ち明けられない環境にいたりしませんか？
一人で悲しみを抱え込み、周りから理解されないことはペットロス症候群を長引かせる原因となります。

悲しみの原因を自分自身で理解しておくことは、気持ちを整理する上で重要なことになります。
なぜ自分がここまで辛い思いをしているのか、自分の心に問いかけてみましょう。

ペットロス症候群を克服するには

自分自身がペットロス症候群になっていたり、周りの人がペットロス症候群になってしまったりする場合はどのような対応をすればよいのでしょうか？

■思いきり泣く

出典 : http://inulog.net/owakare/70.html

泣くことに抵抗を感じて我慢してしまう方もいるようですが、悲しいときには思い切り涙を流して泣きましょう。
悲しいときに流す涙は普段の涙と成分が違い、ホルモンや神経伝達物質が多く含まれているといいます。
悲しみを乗り越えるためにたくさん泣いておくことは必要なことなのです。

■同じ体験をした人と話をする

出典 : http://cocorofull.com/?page_b1154=3

「悲しい経験をしたのは自分だけでない」
「共感できる人がいる」

人間とは不思議なもので、これらのことが確認できるだけで心が落ち着くことがあります。
もし周りにこういう方がいなければ、インターネットなどで体験談を読んだりするのも良いでしょう。

■ペットに手紙を書いてみる

出典 : http://aisis99.blog65.fc2.com

失ってしまったペットへの手紙を書いてみましょう。
手紙を書くことで自分の中の気持ちが整理され、なぜ自分が悲しい気持ちになってしまっているかを確認することができます。
どうしても伝えたかったことを書き、自分の思いを失ったペットに伝えてみましょう。

■ペットとの思い出の品を見えないところに片付ける

出典 : http://nahaten.exblog.jp/m2011-09-01/

家の中をペットがいたときのままにしておくとどうしても思い出すことが多くなります。
「あそこで寝ていたのに…」「あれで遊んでいたのに…」といつまでも引きずる原因になりますので、気持ちを切り替えるためにも思い出の品は見えないところに片付けておきましょう。

■病院でカウンセリングを受ける

出典 : http://iryo.sanyo.oni.co.jp

どうしても立ち直れない場合には心の専門家を頼るのも一つの手段です。
特に自殺の兆候がある場合は、早めに心療内科を受診しましょう。

■悪質なアニマルコミュニケーターには気を付けて

出典 : http://blogs.yahoo.co.jp

「失ったペットの霊を呼ぶ」というアニマルコミュニケーター。
カウンセリング代わりになればよいのですが、中には「あなたのペットには悪い霊がついている」などと言って高額な商品を買わせたりする悪質なものもあります。

ペットロス症候群にならないためにできること

現在ペットを飼っている方に読んでほしい項目です。

■ペット以外の楽しみを見つける

出典 : http://www.0384saisei.com

もしあなたの生活がペットで100％を占めているようなら、改善が必要です。
ペット以外で趣味を作ったり、交友関係を広げてみたりしてはいかがでしょうか？

余裕があれば、もう一頭ペットを飼育して愛情を分散するのも良いでしょう。

■ペットの健康チェックをこまめに行う

出典 : http://nttbj.itp.ne.jp

「もっと早く病気に気付いていれば…」という自責の念はペットロス症候群を引き起こしやすくなります。
ペットの寿命を延ばすためにも、普段から健康チェックをこまめに行っておきましょう。

■不慮の事故が起きないように注意して

出典 : http://blog.hangame.co.jp

「あのときもっと気を付けていれば…」
不慮の事故も自責の念を引き起こし、ペットロス症候群を長引かせる原因となりがちです。

事故はどうしても防げないこともありますが、知識があれば予防できることもあります。
自分の飼っているペットはどんな事故に遭いやすいのか、何に気を付けなければならないのか、今一度勉強しておきましょう。

■亡くなるときにはできるだけ立ち会ってあげて

出典 : http://kabukilily.a-thera.jp

愛する者の死を看取ることは辛いことですが、可愛がったペットの最後にはできるだけ立ち会ってあげるようにしましょう。
どうしても立ち会うことができなかった場合には、死後の対面でもかまいません。

死に目に立ち会えなかったことを後悔する方は少なくないようです。
仕事や学校などあるかもしれませんが、できるだけ立ち会ってあげるようにしたいものです。

最後に

最後に、「虹の橋」という詩をここに載せておきたいと思います。
もしペットロスに悩むことがあれば、ぜひ読んでみてください。
そして、何度も思い返してください。
そこにあなたの可愛かったペットがいます。

天国には「虹の橋」と呼ばれる場所があります。
飼い主にこよなく愛されていた動物は全て、この虹の橋に集まります。

誰もが皆、走り回ったり遊んだり出来るよう、ここには草原や小高い丘があります。
食べ物も飲み水も豊富にあり、太陽が暖かく照り付けてとっても心地よい場所です。

病気や老いで弱っていた動物たちは、全て元の元気な体に戻ります。
傷ついたり体が不自由だった動物たちも、全て健康で活力に満ちた体を取り戻します。

それはちょうど、飼い主がペットが元気だった頃を懐かしむときの姿です。
動物たちは皆満ち足りており、何の不満もありません。

でも、たった一つだけ、気がかりなことがあります。
それは一緒に虹の橋に連れてくることの出来なかった、
飼い主であるあなたのことです。

動物たちはみんな仲良く集まってはしゃぎ回りますが、
そのうちふと足を止めて遠くを見つめるときが来るでしょう。

やがて瞳がらんらんと輝き、体が震えだしたかと思うと、
突然仲間のもとから離れて緑の草原を駆け抜け、
飛ぶように走りに走るのです。

そう、あなたを見つけたのです。

特別な感情で結ばれたあなたと友達は、とうとうこの場所で再会を果たし、
もう二度と別れることのない新たな出会いに胸を躍らせます。

友達はあなたの顔にキスの雨を降らせ、
あなたも懐かしい友達の体を撫でて長いこと見ることの無かった
その瞳を見つめます。

片時も忘れることの無かったその瞳を。

そうしてあなたと友達は、共に虹の橋を渡るのです。
