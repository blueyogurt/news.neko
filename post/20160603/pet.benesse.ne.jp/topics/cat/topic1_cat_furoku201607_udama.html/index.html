猫漫画家うだまさんの愛猫が「イルムスの爪とぎソファ」を使ってみた！

2016.6.3 いぬのきもち/ねこのきもち remove_red_eye13675

(　･ω･)＜こんにちは。猫漫画家のうだまです。『ねこのきもち』2016年７月号のふろく「イルムスの爪とぎソファ」を、ひと足お先に使ってみました。

まずは、組み立てから

(　･ω･)＜ふろくの袋を開けてみると、こんな感じです。これを組み立ててソファにしていきます。

(　･ω･)＜さっそく猫が寄ってきました。

(　･ω･)<潜るつもりの模様。

(　･ω･)<たぶん皆さまのお手元に届く際も、こうやって猫が組み立てを邪魔しに来るかと思います（笑）。それも含めて、組み立ての醍醐味ですネ。

(　･ω･)<説明書を見ながら、組み立てていきます。とは言っても、実に簡単。土台を作り、背もたれを差し込み、これで枠の完成です。

(　･ω･)<最後に、先ほど猫たちがじゃれついていた段ボール製の爪とぎを中に入れて、完成です。表面がボロボロになってきたら、裏返しにすればいいので長く使えますネ。

実際に使ってみた

(　･ω･)＜さっそく、わが家の末っコがやってまいりました。実はこのコ、段ボール製の爪とぎを使うのはこれが初めて……。使い方、わかるかしら。

(　･ω･)<アッカンベーってされたのでこりゃだめっぽいですね（笑）。

(　･ω･)<じゃあ兄猫さんたちに、お手本を見せてもらいましょう！

(　･ω･)＜使い方を目の前で見せてもらって、それで学習すればいいかなと思ったのですが……。

(　･ω･)<お兄ちゃんが使おうとするものは「アタシもあそぶ！」とついてくる末っコの姫様。

(　･ω･)<使い方もまだわかってないのに爪とぎを奪ってしまいました。兄猫の表情にご注目。「マジかよ」って顔してます。

(　･ω･)<新しいグッズに皆さん興味津々。わらわらと猫が集まってきました。

(　･ω･)<トップバッターは一番上の兄猫・リクちゃんです。この子は爪とぎがとにかく下手っぴなので……。

(　･ω･)<ニオイをかいでは肉球の表面をこすりつける、ということを繰り返していました（笑）。そこへ２番目の兄猫・ハルちゃんがやって来て……。

(　･ω･)<爪とぎを、奪取。リクが「あーあ……」という顔をしていますね。

(　･ω･)<そして諦めたのか……

(　･ω･)<立ち去るリクちゃん。そしてその横で夢中で爪とぎをするハルちゃん。

(　･ω･)<ハルちゃんが「お兄ちゃん、なんで爪とぎ使わないのかな？」って不思議そうにリクちゃんを見つめています。キミが独占しているからですよと内心思いながら飼い主はこの光景を眺めます。

もちろんベッドとしても使用可能

(　･ω･)<そしてこちらの爪とぎソファがたいそう気に入った、わが家の王子さま。小柄な体をスッ……と横たえて優雅にくつろいでおりました。

(　･ω･)<こちらのハルちゃんの体重は3.7キログラム。これくらいの大きさの猫さんならば、ジャストサイズで使用できそうです。

(　･ω･)<ちなみに。ソファベッドには、淡い水色で「ねこのきもち」って書いてあります。

(　･ω･)<以上、ソファベッドの使用レビューでございました。一番これを気に入って使ってくれたのは２番目の猫・ハルちゃんでした。

(　･ω･)<そしてオマケ。動画バージョンです。かなりまったりとリラックスしていました。






(　･ω･)<ハルちゃん、これからも思う存分、使ってくださいね！
